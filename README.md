# MagicBox homework
## Setup Enviroment
1. Create a new virtual environment:
```shell
python -m venv myenv
```
2. Activate the virtual environment:
On Windows:
```shell
myenv\Scripts\activate.bat
```
On macOS and Linux:
```shell
source myenv/bin/activate
```
